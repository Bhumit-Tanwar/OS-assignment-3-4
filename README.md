# 🚀 Assignment 3: Page Replacement Algorithms

## 📖 Description
This assignment implements different Page Replacement Algorithms used in operating systems to manage memory efficiently.

## 🧠 Algorithms Implemented
- FIFO (First-In First-Out)  
- LRU (Least Recently Used)  
- Optimal  
- MRU (Most Recently Used)  
- Second Chance (Clock Algorithm)  

## 🎯 Objective
- Understand virtual memory and paging  
- Calculate page faults  
- Compare algorithm performance  

## 📥 Input
- Number of Frames  
- Page Reference String  

## 📤 Output
- Total Page Faults  

## 📸 Output Screenshot


## 📊 Observations
- Optimal gives minimum page faults  
- LRU performs better than FIFO  
- Second Chance improves FIFO  
- MRU is less efficient
